Working of a Shop Website on Internet & Hosting

1. What is a Shop Website?

A shop website is a collection of web pages that displays products, prices, and shop information online.

Example: A grocery shop website showing rice, milk, fruits, and contact details.

2. How a Shop Website Works on the Internet

  1. User enters website URL in browser
  2. Browser sends request to web server
  3. Server sends HTML, CSS, and JavaScript files
  4. Browser displays the website

Example: User opens www.freshmart.com and product list is shown.

3. Role of HTML, CSS, and JavaScript

  • HTML: Creates structure of website
  • CSS: Adds design, colors, and layout
  • JavaScript: Adds interactivity and dynamic behavior

Example: Clicking an “Add to Cart” button.

4. What is a Web Server?

A web server is a computer that stores website files and sends them to users on request.

Example: Apache, Nginx, AWS server.

5. What is Hosting?

Hosting means storing website files on a server so the website is available on the Internet 24×7.

Example: Hosting website on GitHub Pages or Netlify.

6. Types of Hosting

  • Shared Hosting
  • VPS Hosting
  • Cloud Hosting

Example: Small grocery shop uses shared hosting.

7. Domain Name

A domain name is the address of a website on the Internet.

Example: www.freshmartgrocery.com

8. Steps to Host a Website

  1. Create website using HTML, CSS, and JavaScript
  2. Buy a domain name
  3. Choose a hosting provider
  4. Upload website files
  5. Website becomes live

9. How Customers Access the Website

Customers use mobile phones, laptops, or tablets and open the website using a browser.

10. Advantages of Shop Website

  • 24×7 availability
  • More customers
  • Online presence
  • Easy product display

Conclusion

A shop website works by sending files from a web server to the user’s browser through the Internet. Hosting makes the website available online.
